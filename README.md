## Getting Started

### Prerequisites

**Node version 14.x**

### Cloning the repository

```bash
git clone https://github.com/7kevinzong/airbnb-pretty-much.git
```

### Install packages

```bash
npm i
```

### Setup .env file

```bash
DATABASE_URL=
GITHUB_ID=
GITHUB_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXTAUTH_SECRET=
```

### Setup Prisma

```bash
npx prisma db push
```

### Start the app

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
